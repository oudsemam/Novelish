About Us

Welcome to Novelish - the not so novel idea for book lovers everywhere!

Novelish is the product of three book lover full stack graduates of Grand Circus, based in West Michigan, who hadn't yet found a satisfying solution for a book organizing app. We wanted a great user experience that was simple and easy to use, and allowed us to save and organize our book lists. Be sure to check back for new features currently in development, like creating your own shelves, advanced searching, tracking progress in your books, and adding notes.

    Technologies Used:

  • Open Library API
  • New York Times Best Sellers API
  • Backend database created in pgAdmin and Insomnia using Express framework
  • Firebase user authentication
  • Sentry error tracking
  • Angular 12 platform
